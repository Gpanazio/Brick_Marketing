# DIREÇÕES CRIATIVAS: CAMPANHA 180 SEGUROS 2026

## Contexto do Pedido
- **Formato:** Vídeo institucional 90-120s + cortes (30s, 15s, 6s)
- **Budget:** R$ 40.000
- **Recursos disponíveis:** Banco de imagens interno + 1 diária de captação
- **Lançamento:** 14 de abril
- **Narrativa central:** "Por que 180?" (escolha inevitável)

---

## CONCEITO A: "A ESCOLHA ÓBVIA"

### Premissa
O mercado de seguros é travado por burocracia, lentidão e distância. A 180 não é uma alternativa — é o que faz sentido. Quando você conhece, não tem como escolher diferente.

### Estrutura Narrativa (90s)
```
0-15s   → O problema: seguro como ele sempre foi (imagens genéricas de burocracia, espera, papelada)
15-30s  → A pergunta: "E se existisse outro jeito?"
30-60s  → A 180: agilidade real (implantação), proximidade real (atendimento), tech real (integração)
60-80s  → Resultados: parceiros que cresceram com a 180 (dados/logos)
80-90s  → CTA: "180. A escolha óbvia."
```

### Tom
Confiante sem arrogância. "Não estamos tentando te convencer. Estamos te mostrando."

### Execução
- Banco de imagens para "mercado tradicional" (genérico, cinza)
- Material interno 180 para "nossa realidade" (colorido, dinâmico)
- 1 diária para depoimentos rápidos de lideranças (opcional)

### Risco
Pode soar como "somos melhores que todo mundo" se não for bem calibrado.

---

## CONCEITO B: "180 GRAUS"

### Premissa
O nome já diz: viramos o jogo. Mas não é sobre destruir o antigo — é sobre fazer diferente, do jeito certo. Uma virada de perspectiva, não de agressividade.

### Estrutura Narrativa (90s)
```
0-20s   → Metáfora visual: câmera girando 180° (de costas → de frente)
20-40s  → Locução: "No mercado de seguros, todo mundo olha pro mesmo lado. A gente escolheu olhar pro cliente."
40-70s  → Pilares: Agilidade (tempo de implantação), Proximidade (atendimento humano), Tecnologia (integração via API)
70-85s  → Frase-âncora: "Não é sobre ser novo. É sobre ser melhor."
85-90s  → Logo + assinatura: "180 Seguros. A nova perspectiva."
```

### Tom
Metafórico no início, direto no meio, memorável no fim.

### Execução
- Motion graphics para a metáfora do giro
- Banco de imagens + material interno para pilares
- Sem necessidade de captação nova (economia de budget)

### Risco
A metáfora do "giro" pode parecer clichê se não for bem executada visualmente.

---

## CONCEITO C: "QUEM JÁ CONHECE, NÃO VOLTA"

### Premissa
Prova social como espinha dorsal. Em vez de a 180 falar de si, deixar os parceiros falarem. A credibilidade vem de quem já está dentro.

### Estrutura Narrativa (90s)
```
0-10s   → Tela preta, texto: "O que nossos parceiros dizem?"
10-50s  → 3-4 depoimentos curtos (10s cada): fintechs, cooperativas, bancos (podem ser em áudio + texto animado se não houver captação)
50-70s  → Dados de resultado: tempo de integração, NPS, crescimento de parceiros
70-85s  → Locução: "Mais de X parceiros já escolheram a 180. Você é o próximo?"
85-90s  → CTA + logo
```

### Tom
Testemunhal, humano, baseado em evidência.

### Execução
- Se tiver depoimentos gravados: usar
- Se não: áudio + animação tipográfica (mais barato)
- 1 diária pode ser usada para captar 3-4 falas de parceiros via Zoom/remoto

### Risco
Depende de ter parceiros dispostos a falar. Se não tiver, enfraquece.

---

## RECOMENDAÇÃO BRICK

**Conceito B ("180 Graus")** é o mais seguro para o budget e prazo:
- Não depende de terceiros (parceiros)
- Aproveita o nome da marca como mote criativo
- Motion graphics escala bem para os cortes (30s, 15s, 6s)
- Tom alinhado com o manual de branding (moderno, confiante, sem hype)

**Conceito C** seria o mais forte em credibilidade, mas exige coordenação de depoimentos — viável se a 180 já tiver material ou conseguir gravar rápido.

**Conceito A** funciona, mas é o mais "comum" — várias seguradoras usam a narrativa de "somos diferentes".

3. Se Conceito C: mapear parceiros disponíveis para depoimento
4. Desenvolver roteiro completo da direção escolhida
